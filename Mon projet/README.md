# Mon Projet

Un projet web simple composé de HTML, CSS et JavaScript. Ce projet est initier à l'origine afin de aidé les personne ayant des difficulter dans des demarche administrative fastidieuse.

Sur ce site nous pouvons retrouver diférente catégorie de paperasse à renmplire. Nous allons voir ci-dessous les diferente cathégorie que nous avons fait.

## Structure du projet

- `index.html` : Le fichier HTML principal.
- `css/style.css` : Les styles CSS.
- `js/script.js` : Le script JavaScript.

## Utilisation

Ouvrez le fichier `index.html` dans votre navigateur web pour voir le projet.

## Prérequis

Aucun prérequis particulier. Un navigateur web moderne suffit.

## Dépannage

Si le projet ne s'affiche pas correctement, vérifiez que tous les fichiers sont présents et que les chemins relatifs sont corrects.
